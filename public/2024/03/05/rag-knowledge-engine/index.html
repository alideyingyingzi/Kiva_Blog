支持的知识领域

领域 说明 适用场景
ABAQUS 有限元分析软件 结构仿真、非线性分析、子程序开发

大模型开发 AI 开发技术 Prompt、RAG、Agent 开发

使用说明

  1. 选择知识领域：点击上方的领域按钮
  2. 配置 API：点击右上角”API 配置”，输入 API 密钥
  3. 输入问题：在文本框中输入您的问题
  4. 获取答案：点击搜索按钮

注意事项

当前状态：直连大模型模式
系统直接调用通义千问 (Qwen) 大模型 API，无需后端服务。
每个知识领域都有专门的 Prompt 模板，确保回答的专业性。
